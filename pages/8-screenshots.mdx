### Alternativas e Imagens

- MDWEB
  - Portal com vários serviços para dinâmica molecular
  - Não suporta simulações de proteínas complexadas
  - Preparação manual de arquivos para simulação

- WebGRO
  - Suporta simulações de proteínas complexadas
  - Apenas PRODRG
  - Apenas campos de força GROMOS

- Plugins para Softwares já existentes
  - QwikMD para VMD
  - GROMACS para PyMOL
  - Opção de simulação de dinâmica molecular para o UCSF Chimera

<!--
- VMD é um programa de computador (instalável) que serve para analisar, exibir e animar grandes sistemas biomoleculares

- PyMOL é um sistema para visualização molecular

- Chimera é um programa para visualização interativa e análise de estruturas moleculares e dados relacionados
