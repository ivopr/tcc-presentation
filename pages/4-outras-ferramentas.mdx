### Outras ferramentas

- ##### Grace: Ferramenta de plotagem 2D para o Sistema de Janelas X.
  - ###### Geração de gráficos para analises de simulações.
- ##### Celery: Fila de Tarefas Distribuida.
  - ###### Execução de simulações de forma controlada e distribuida.
- ##### Flask: Framework Web para Python
  - ###### Codificação do _backend_ em python para execução e geração de simulações.
- ##### NextJS: Framework React para a Web
  - ###### Codificação do _frontend_, possibilitando a disponibilização de uma interface intuitiva e amigável.
